# AI & Pharma Regulatoion Database

This database maps international laws, regulations, and formal guidance documents relevant to the deployment of Artificial Intelligence and Machine Learning across the pharmaceutical lifecycle.
The tsv and xlsx files have identical content to the table below but are provided for convenience

> **⚠️ PROVISIONAL DATABASE NOTICE**  
> This database is an **provisional resource** (version 0) currently under active curation. Due to the rapidly evolving nature of global AI policies, digital health laws, and pharmaceutical guidelines, some entries may undergo frequent updates.

---

## 🤝 How to Contribute

We welcome contributions from regulatory affairs experts, AI engineers, legal scholars, and policy enthusiasts! If you want to expand this matrix, add new jurisdictions, or correct an outdated link, please follow these steps:

### 1. Suggest a Modification or Addition
* **Submit an Issue:** Click on the **Issues** tab above and select **New Issue**. Use this to point out broken URLs, outdated legislation, or to request the addition of a new country.
* **Submit a Pull Request (PR):** 
  1. Fork this repository.
  2. Update the tsv file directly in your fork.
  3. Submit a PR back to the `main` branch with a description of what changed (e.g., *"Updated Health Canada link"* or *"Added Singapore LLM clinical guidance"*).

### 2. Standard Taxonomy for New Entries
To keep the database highly structured and scannable, please ensure all new entries map strictly to our 5 core regulatory pillars and use the following database columns exactly:

* **RegulatoryWeight:** `Mandatory`, `Guidance`, `Industry Standard`
* **LegalDomain:** `General AI/Data/Software`, `General Pharma`, `Digital Health Law`, `AI Ethics & Governance`, or `AI/Data/Software+Pharma Intersection`
* **LifecycleDomain:** `Discovery / Preclinical`, `Clinical`, `Regulatory Review`, `Manufacturing`, `Post-Market`, or `Full Lifecycle`

---

## Content Description
All entries in this database align with one or more of the following critical pillars:
1. **Medicines / Drug Law:** Core legal basis for pharmaceutical market entry.
2. **Clinical Trial Regulation:** Governance of human subject research and data collection.
3. **Medical Device / SaMD Regulation:** The critical validation layer for clinical algorithms.
4. **Data Protection / Privacy Law:** Legal frameworks for handling patient, genomic, and employee data.
5. **Post-Market Surveillance / Pharmacovigilance:** Continuous safety, data-drift, and signal monitoring.
